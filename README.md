# MIN Mobile App

Open-source AI-free mobile application for the MIN messaging platform written in React Native.

## Settings

### .env configuration:

-   **`SERVER`** - string; server URL (e.g. https://api.example.com)
-   `VERSION` - string; version code (e.g. 1.0.0)
-   `AUTO_UPDATE` - boolean; enable or disable auto updates (e.g. true)
-   `REPO_AUTHOR` - string; used for auto update
-   `REPO_NAME` - string; used for auto update
